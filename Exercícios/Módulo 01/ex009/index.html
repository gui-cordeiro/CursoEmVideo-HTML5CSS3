Trabalhando com Listas

Listas Ordenadas

Para criar listas ordenadas, utilize a tag <ol> (Ordered Lists), seguidos das tags <li> (List Item), que correspondem aos itens da lista. Exemplo:

1) Lista de coisas para fazer:

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes
  5. Ir para a faculdade
  6. Almoçar
  7. Ir pro trabalho
  8. Voltar para casa
  9. Jantar
  10. Dormir

Listas Não Ordenadas

Para criar listas não ordenadas, utilize a tag <ul> (Unordered Lists), seguidos das tags <li> que correspondem aos itens da lista. Exemplo:

1) Lista de compras do supermercado:

  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão
