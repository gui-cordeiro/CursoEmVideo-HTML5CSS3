Formatações do HTML

Principais Formatações

Negrito / Destaque

Nessa frase temos um "termo em negrito" usando a tag <b> (não semântica).

Nessa frase, tenho um "termo em destaque" usando a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase, temos um "termo em itálico" usando a tag <i> (não semântica).

Já nesta frase, temos um "termo em ênfase" usando a tag <em>, de "emphasis" (semântica).

Sublinhado / Inserido

 Podemos marcar um texto com sublinhado com a tag <u> (underline, não semântico).

Além disso, podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois com a tag <ins> (semântico).

Outras Formatações

Texto Marcado (Estilo "marca-texto")

Podemos criar também um texto marcado usando a tag <mark>.

E no outro parágrafo, temos um outro texto marcado no final, também com a tag <mark>.

Texto Grande e Pequeno

Estamos criando um texto grande (tag <big>) e um texto pequeno(tag <small>) nesse parágrafo.

Texto Deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado, com a tag <del>.

Texto Sobrescrito

Para inserir coisas do tipo "ax2 + bx + c = 0", utiliza-se a tag <sup>.

Texto Subscrito

Para inserir coisas do tipo "C6H12O6", utiliza-se a tag <sub>.
