Código-fonte

Para escrever trechos de código-fonte de maneira adequada no HTML5, utilize a tag <code>. Exemplo:

1) O comando document.getElementById('teste') é escrito em linguagem JavaScript.

2) Analise o seguinte código-fonte escrita em Linguagem C:

            
    int main(void){
        int num = 0;
        printf("Digite um número: ");
        scanf("%d", &num);
        fflush(stdin);
        if (num % 2 == 0) {
            printf("O número %d é PAR", num);
        } else {
            printf("O número %d é ÍMPAR", num);
        }
    }

Citações

Para descrever citações simples, utilize a tag <q>. Exemplo:

1) Como diria o pai de um amigo: o computador é um burro muito rápido.

Citações Completas

Para descrever citações completas(maiores), utilize a tag <blockquote>. Exemplo:

1) Segundo Jeff Noble, no seu livro HTML para leigos:

Pense nesse livro como um guia amigável e acessível para dominar (X)HTML e CSS e criar páginas web legíveis e atraentes. Essas coisas não são difíceis de aprender, mas envolvem muitos detalhes com os quais você deve lidar conforme constrói suas próprias páginas web.

Abreviações

Para criar descrições de abreviações, utilize a tag <abbr>. Exemplo:

1) Estou estudando HTML e CSS. Estou adorando.

Texto invertido

Para escrever um texto de maneira invertida, utilize a tag <bdo>. Exemplo:

1) Estou aprendendo a criar coisas em HTML. (Estou aprendendo a criar coisas em HTML.)
